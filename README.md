# Marketing Campaign Performance Analysis

## Objective
Evaluate ROI and performance of digital marketing campaigns.

## Tools Used
- Advanced Excel
- Pivot Tables
- Data Visualization

## Key Insights
- Google Ads generated highest conversion rate.
- Facebook showed moderate CTR.
- Budget reallocation suggested for underperforming channels.

- Campaign	   Channel	Impressions 	Clicks	Conversions	Cost
C1	Facebook	10000	     800	          50	             20000
C2	Google Ads	15000	   1200	          90	         35000
C3	Instagram	  12000	     700	          40	             18000

Add calculated columns:

CTR = Clicks / Impressions
Conversion Rate = Conversions / Clicks
ROI = (Revenue - Cost) / Cost
