# Credit Card Sales Dashboard — Power BI Project

## Overview

This Power BI dashboard project analyzes credit card sales performance using interactive visualizations and KPI tracking.

## Features

- KPI Cards
- Bank-wise Analysis
- Sales Trend Analysis
- Geographic Analysis
- Customer Rating Analysis
- Interactive Slicers
- Power Query Transformations
- DAX Measures
#Tools Used
- Power BI
- Excel
- DAX
- Power Query
- Python
- Google Colab

## Dataset

The dataset contains 12,000 customer-level sales records.

## Key Insights
- SBI and HDFC generated the highest sales.
- Customer ratings remained stable across banks.
- Major cities contributed most to total revenue.
- Positive customer sentiment exceeded negative feedback.
